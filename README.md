# funcao13
Criar uma função que recebe dois parâmetros:
- Número de linhas N
- Número de colunas M
e retorna uma matriz NxM, onde cada posição recebe o valor de i elevado a j, onde i é a linha e j é a coluna
- Exemplo para N = 5 e M = 4

Mat:

    1  0  0  0
    1  1  1  1
    1  2  4  8
    1  3  9  27
    1  4  16 64
